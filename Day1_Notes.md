# Day 1 – Network Basics

## What is a Network?
A network is a group of devices connected to share data.

## Types of Networks
- LAN (Local Area Network)
- WAN (Wide Area Network)
- MAN

## What is the Internet?
The Internet is a global network of interconnected networks.

## Key Concepts Learned
- IP Address
- Router
- Switch
- DNS

## Real Life Example
When I open Google:
1. My device sends request
2. DNS converts domain to IP
3. Server responds with data

## My Understanding

Today I learned the basic idea of a network. A network is when multiple devices are connected together to share data and communicate with each other.

Networks can be of different sizes:
- Small networks are called LAN (Local Area Network), like in a home, school, or office.
- Large networks are called WAN (Wide Area Network), which connect devices across cities or countries.
- MAN (Metropolitan Area Network) covers a larger area like a city.

I also understood what the Internet is. The Internet is basically a huge network of networks that connects devices all around the world. It allows us to access websites, send messages, and share information globally.

## Key Concepts (In Simple Words)

- IP Address:
  It is like a home address for your device. Every device connected to the internet has a unique IP address so it can send and receive data.

- Router:
  A router works like a traffic manager. It connects different devices to the internet and directs data to the correct destination.

- Switch:
  A switch connects multiple devices within the same network (like in an office) and helps them communicate efficiently.

- DNS (Domain Name System):
  DNS is like a contact list of the internet. Instead of remembering IP addresses, we use names like google.com, and DNS converts them into IP addresses.
